*An engineering web application for designing standalone photovoltaic (PV) systems.*

*(GitHub Pages URL will be added once the project is published.)*

--------------------------------------------------

## 🎯 Project Overview

The Off-Grid Solar System Calculator is a web application that helps users estimate the size of an off-grid photovoltaic (PV) system based on their electrical load requirements. Users will be able to calculate recommended solar array size, battery capacity, battery autonomy, and the number of solar panels needed using common engineering formulas.

I began building this project to combine my interests in electrical engineering and software development while expanding on my undergraduate research experience designing an off-grid solar power system for transportation infrastructure. I also wanted to learn and strengthen my HTML/CSS and Javascript skills. The project also serves as a way to learn front-end web development and create a practical engineering tool from scratch.
